# Bernard 

![alt text](http://68.media.tumblr.com/avatar_4a9eac096ae1_128.png) 
* Make a directory 
```
mkdir Bernard
```
* change into the directory
```
cd Bernard
```
* Make a directory inside the Bernard directory 
```
mkdir build
```
* change into build
```
cd build
```
* CMake 
``` 
cmake .. 
``` 
* Make 
```
make 
```
* run Bernard 
```
./Bernard 
```
